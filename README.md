# 🦷 AurSmile — Эстетичный Landing Page стоматологической клиники

AurSmile — это проект фронтенд-разработки, ориентированный на создание легкого, чистого и вызывающего доверие интерфейса для медицинских услуг. Основной упор сделан на визуальную иерархию и удобство взаимодействия пользователя с контентом.

## Технический стек и решения
При разработке данного проекта я сфокусировался на семантике и чистоте архитектуры:

* **Semantic HTML5:** Использование правильной структуры тегов для улучшения SEO и доступности (Accessibility).
* **CSS Architecture:** Модульный подход к стилям с использованием Flexbox для точного позиционирования элементов.
* **JS Interactions:** Реализация интерактивных компонентов (меню, скролл-эффекты) на чистом JavaScript.
* **Hybrid Workflow:** Использование инструментов искусственного интеллекта для генерации и оптимизации логических блоков кода, что позволило добиться высокой чистоты и читаемости структуры.

## Что реализовано в проекте
* **Clean UI:** Минималистичный дизайн в пастельных тонах, подчеркивающий профессионализм клиники.
* **Interactive Services:** Динамические блоки услуг с плавными эффектами при наведении (hover effects).
* **Adaptive Grid:** Интерфейс полностью перестраивается под мобильные устройства, сохраняя удобство кликов и чтения.
* **Optimized Assets:** Грамотное использование изображений и шрифтов для обеспечения быстрой отрисовки страницы (LCP).

## Основные цели
Главной целью было создать интерфейс, который не перегружает пользователя лишней информацией, а ведет его к целевому действию — записи на консультацию через интуитивно понятную форму.
