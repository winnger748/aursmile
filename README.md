# 🦷 AurSmile — Эстетика и функциональность в медицинском интерфейсе

AurSmile — это пример того, как дизайн и код работают на создание атмосферы спокойствия и профессионализма. Основной вызов проекта заключался в достижении визуальной легкости при сохранении строгой технической структуры.

## 🛠 Методология разработки (Iteration Process)
Проект прошел через цикл итераций для достижения эталонного качества:
1. **Проектирование UX:** Разработка логики блоков, ориентированная на психологию пациента (доверие -> услуги -> запись).
2. **Модульная архитектура:** Разделение проекта на чистые слои (HTML, CSS, JS). Это позволило сделать код прозрачным и легко поддерживаемым.
3. **SEO-интеграция:** Внедрение расширенных метаданных для обеспечения видимости проекта в поисковых системах.

## ⚙️ Технические решения
В AurSmile я реализовал профессиональный подход к фронтенд-разработке:
* **Separation of Concerns:** Логика, стили и структура полностью разделены. Это стандарт индустрии, который я строго соблюдаю.
* **Typography & Spacing:** Работа с "воздухом" (whitespace) через модульные сетки Flexbox и CSS Grid.
* **JS Performance:** Интерактивные элементы написаны на чистом JS с использованием паттерна делегирования событий для оптимизации нагрузки на браузер.
* **Core Web Vitals:** Оптимизация LCP (Largest Contentful Paint) через правильное подключение шрифтов и минимизацию блокирующих рендеринг ресурсов.
* **Semantic & Accessibility:** Использование правильных HTML5 тегов для доступности (A11y) и SEO-оптимизации.

## 🎯 Результат
* **Clean & Modular Code:** Проект легко масштабируется благодаря четкой структуре файлов.
* **High Visibility:** Сайт полностью подготовлен к индексации и корректному отображению в соцсетях (Open Graph).
* **Full Adaptivity:** Бесшовная работа на любых устройствах — от смартфонов до широкоформатных мониторов.
