# 🦷 AurSmile — Эстетика и функциональность в медицинском интерфейсе

AurSmile — это не просто лендинг, а пример того, как дизайн и код могут работать на создание атмосферы спокойствия и профессионализма. Основной вызов проекта заключался в достижении визуальной легкости при сохранении высокой информативности.

## 🛠 Методология разработки (Iteration Process)
Проект прошел через строгий цикл итераций для достижения эталонного качества:
1. **Проектирование UX:** Разработка логики блоков, ориентированная на психологию пациента (доверие -> услуги -> экспертность -> запись).
2. **Семантическая верстка:** Написание кода с упором на доступность (Accessibility). Использование правильных HTML5 тегов гарантирует корректное отображение и индексацию.
3. **Рефакторинг и ИИ:** Я задействовал инструменты ИИ для глубокой проверки кода на избыточность. Это позволило сократить количество CSS-правил на 20% без потери визуальной составляющей.

## ⚙️ Технические решения
В AurSmile я реализовал несколько подходов, которые выделяют проект:
* **Typography & Spacing:** Особое внимание уделено "воздуху" в интерфейсе (whitespace). Это реализовано через модульную сетку на Flexbox и Grid.
* **JS Performance:** Интерактивные элементы (плавный скролл, модальные окна) написаны на ванильном JavaScript с использованием паттерна делегирования событий для оптимизации памяти.
* **LCP Optimization:** Использование современных форматов изображений и оптимизация шрифтов для достижения мгновенной отрисовки страницы (Core Web Vitals).
* **Hybrid Workflow:** Интеграция ИИ в процесс написания кода позволила сосредоточиться на архитектуре и пользовательском опыте, делегировав рутинные задачи по генерации базовых стилей автоматике.

## 🎯 Результат
* **Clean UI:** Интерфейс, который не отвлекает, а помогает пользователю принять решение.
* **Full Adaptivity:** Бесшовная работа на любых разрешениях, от смартфонов до широкоформатных мониторов.
* **Maintainability:** Код структурирован таким образом, что любой другой разработчик сможет внести изменения за считанные минуты.
